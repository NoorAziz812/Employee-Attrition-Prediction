# 📊 Employee-Attrition-Prediction

Machine learning project for predicting employee attrition using Logistic Regression with L1 and L2 regularization.

## 🎯 Overview
Predictive analytics solution designed to identify employees at risk of leaving the organization using machine learning models.

## 📁 Project Structure
```
├── data/              # 📊 Dataset files
├── models/            # 🤖 Trained models
├── notebooks/         # 📓 Jupyter notebooks
├── scripts/           # 🔧 Python scripts
└── README.md         # 📖 Documentation
```

## 🔧 Technologies & Tools
- **Language**: 🐍 Python
- **Machine Learning**: 📚 scikit-learn
- **Data Analysis**: 🔬 pandas, NumPy
- **Visualization**: 📈 Matplotlib, Seaborn
- **Regularization**: ⚖️ L1 (Lasso) & L2 (Ridge)

## 📊 Model Details
- **Algorithm**: 🎯 Logistic Regression
- **Regularization**: L1 and L2
- **Input Features**: 🔍 Employee demographics, performance metrics, tenure
- **Output**: ✅/❌ Attrition prediction (Binary classification)

## 📈 Key Features
- ✨ Data preprocessing and feature engineering
- 🔬 Model training with cross-validation
- 📊 Performance evaluation (Accuracy, Precision, Recall, F1-Score)
- 📉 Comparative analysis of L1 vs L2 regularization
